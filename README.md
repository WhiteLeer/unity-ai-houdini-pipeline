# unity-ai-houdini-pipeline

面向 Unity 内容生产的自动化流水线：将图像生成、模型处理与 DCC 优化串联为可重复执行的工程流程。

## 项目简介

本仓库聚焦于三件事：

- 在 Unity 编辑器内组织长链路任务
- 对接外部 3D 生成服务
- 通过 Houdini 脚本完成模型优化与落地

## 目录结构

- `Assets/`：Unity 侧脚本与编辑器工具
- `Houdini/`：Houdini 自动化脚本
- `docs/`：补充说明文档

## 使用前提

- Unity 2019.4 及以上
- 可调用 `hython` 的 Houdini 环境
- 已配置可用的第三方服务密钥

## 使用方式

1. 将仓库中的 Unity 脚本导入项目。
2. 在工具面板中填写服务凭据与 Houdini 可执行路径。
3. 在编辑器内触发流水线，完成从输入到资产输出的处理。
